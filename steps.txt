Installing node (https://tecadmin.net/how-to-install-nvm-on-ubuntu-20-04/):
sudo apt install curl 
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
source ~/.bashrc
nvm install <node_version>


Installing docker fastify (https://medium.com/@davidkelley87/fastify-docker-%EF%B8%8F-9165da21ae23):
mkdir fastify-docker
cd fastify-docker
npm init fastify -y
npm i -y


Dockerfile content:
FROM node:18

WORKDIR /app

COPY package.json .

RUN npm install -y

COPY . .

EXPOSE 3000

CMD ["npm", "start"]


useful docker commands:
sudo docker ps
sudo docker stop <container_id>
sudo docker rm <container_id> or <docker_name>
sudo docker images
sudo docker rmi <image_id> -f
